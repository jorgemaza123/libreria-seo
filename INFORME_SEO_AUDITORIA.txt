RESUMEN EJECUTIVO
-----------------
Se ha realizado una auditoria exhaustiva del sitio web "Libreria Central".
El proyecto esta construido con Next.js 16.1.3 y React 19.2.3, con una base
de SEO bien establecida pero con areas criticas que requieren atencion
inmediata para optimizar el posicionamiento en Google.

ESTADO GENERAL: 65/100 (Necesita mejoras importantes)

================================================================================
                         ERRORES CRITICOS (Prioridad Alta)
================================================================================

1. NO EXISTE ROBOTS.TXT
   Ubicacion esperada: /public/robots.txt o /src/app/robots.ts
   Estado: NO ENCONTRADO
   Impacto: Google no tiene instrucciones claras sobre que indexar
   Solucion: Crear archivo robots.txt con directivas apropiadas

2. NO EXISTE SITEMAP.XML
   Ubicacion esperada: /public/sitemap.xml o /src/app/sitemap.ts
   Estado: NO ENCONTRADO
   Impacto: Google no puede descubrir todas las paginas eficientemente
   Solucion: Implementar sitemap dinamico con Next.js

3. IMAGEN OG (Open Graph) NO EXISTE
   Referencia en codigo: /og-image.jpg
   Ubicacion: src/lib/seo.ts linea 19
   Estado: ARCHIVO NO ENCONTRADO en /public/
   Impacto: Compartir en redes sociales no mostrara imagen
   Solucion: Crear imagen de 1200x630px optimizada

4. URL BASE NO CONFIGURADA (Placeholder)
   Archivo: src/lib/seo.ts linea 16
   Valor actual: "https://libreria-central.com"
   Estado: DOMINIO NO CONFIGURADO/ACTIVO
   Impacto: Todas las URLs canonicas apuntan a dominio inexistente
   Solucion: Actualizar cuando se tenga dominio final

5. NUMEROS DE WHATSAPP INCONSISTENTES
   - HeroSection.tsx: 51987654321 (lineas 108, 174, 183)
   - mock-data.ts: 51999999999 (linea 261)
   - Footer usa mock-data (correcto)
   Impacto: Confusion del usuario, posible perdida de ventas
   Solucion: Unificar todos los numeros de contacto

================================================================================
                         ERRORES IMPORTANTES (Prioridad Media)
================================================================================

6. LOCALBUSINESS SCHEMA NO INYECTADO
   Archivo: src/lib/seo.ts (lineas 25-75)
   Estado: Funcion definida pero NUNCA SE USA en el HTML
   Impacto: Google no puede identificar el negocio local
   Solucion: Inyectar schema JSON-LD en layout.tsx o page.tsx principal

7. FAQ SCHEMA NO IMPLEMENTADO
   Archivo: src/lib/seo.ts (lineas 110-121)
   Estado: Funcion definida pero NO SE USA
   Impacto: Las FAQs no aparecen como rich snippets en Google
   Solucion: Inyectar FAQSchema en pagina donde se muestran las FAQs

8. BREADCRUMB SCHEMA INCOMPLETO
   Archivo: src/app/producto/[slug]/page.tsx (lineas 79-99)
   Estado: Breadcrumbs visuales pero SIN JSON-LD
   Impacto: Google no entiende la estructura de navegacion
   Solucion: Agregar BreadcrumbSchema JSON-LD junto al visual

9. LINKS DEL FOOTER ROTOS
   Archivo: src/components/layout/Footer.tsx (lineas 177-182)
   - "Aviso de Privacidad" apunta a href="#"
   - "Terminos y Condiciones" apunta a href="#"
   Impacto: Enlaces rotos afectan experiencia y SEO
   Solucion: Crear paginas legales o remover enlaces

10. FALTA PRECONNECT PARA DOMINIOS EXTERNOS
    Dominios usados sin preconnect:
    - images.unsplash.com (imagenes de productos)
    - fonts.googleapis.com (fuentes)
    - fonts.gstatic.com (fuentes)
    Archivo: src/app/layout.tsx
    Impacto: Retraso en carga de recursos externos
    Solucion: Agregar tags <link rel="preconnect"> en head

================================================================================
                         ADVERTENCIAS (Prioridad Baja)
================================================================================

11. NO HAY MANIFEST.JSON (PWA)
    Estado: No existe archivo manifest para Progressive Web App
    Impacto: La app no se puede instalar en dispositivos moviles
    Solucion: Crear manifest.json con metadata de la app

12. NO HAY PAGINAS DE CATEGORIAS INDIVIDUALES
    Estado actual: Solo existe /#productos?category=slug
    Impacto: Menos oportunidades de rankear por categoria
    Solucion: Crear /categoria/[slug] con contenido unico

13. NO HAY PAGINAS DE SERVICIOS INDIVIDUALES
    Estado actual: Solo seccion en homepage
    Impacto: Perdida de oportunidades de ranking por servicio
    Solucion: Crear /servicio/[slug] con contenido detallado

14. ALT TEXTS DE IMAGENES GENERICOS
    Ejemplo: ProductsSection.tsx linea 27
    alt={product.name} - Usa solo el nombre
    Mejor practica: "Mochila Escolar Premium - Libreria Central SJL"
    Impacto: Menor relevancia en Google Images

15. FALTA CANONICAL URL EXPLICITA EN PRODUCTOS
    Archivo: src/app/producto/[slug]/page.tsx
    Estado: No hay canonical definido en metadata
    Impacto: Posible contenido duplicado si hay parametros URL

16. NO HAY HREFLANG TAGS
    Estado: Sitio solo en espanol-Peru
    Impacto: Bajo (solo relevante si hay multiples idiomas)
    Nota: Considerar si se expandira a otros paises
